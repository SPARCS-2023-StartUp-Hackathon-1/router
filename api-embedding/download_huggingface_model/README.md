# Download Huggingface Pretrained Model

## How to use

- download packages

  ```bash
  $ pip3 install -r requirements.txt
  ```

- save huggingface pretrained-model

  ```bash
  $ python3 downloadAndSave.py <pretrained-model-name>
  ```

- load saved model (for example)

  ```bash
  $ python3 load.py
  ```
